# 📊 Telecom Customer Churn Prediction using Deep Learning

This project predicts customer churn in a telecommunications company using an Artificial Neural Network (ANN) built with TensorFlow/Keras.

## 🎯 Objective

The goal is to identify customers who are likely to leave the company so that retention strategies can be implemented proactively.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- Matplotlib
- Seaborn

## 📋 Project Workflow

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding and Scaling
- Handling Class Imbalance using Class Weights
- ANN Model Development
- Overfitting Prevention using Dropout and Early Stopping
- Model Evaluation

## 🧠 Model Architecture

- Input Layer (30 Features)
- Dense Layer (16 Neurons, ReLU)
- Dropout (0.2)
- Dense Layer (8 Neurons, ReLU)
- Dropout (0.2)
- Output Layer (1 Neuron, Sigmoid)

## 📈 Results

| Metric | Score |
|----------|----------|
| Accuracy | 71% |
| Recall (Churn) | 81% |
| Precision (Churn) | 48% |
| F1-Score (Churn) | 60% |

### Key Outcome

Successfully identified **81% of customers at risk of churn**, helping support proactive customer retention strategies.
